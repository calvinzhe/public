Using the Jupyter Notebook in a Mathematica-like way¶

Setting up Jupyter Notebook¶

This setup uses the IPython kernel, but Jupyter Notebook can be used to run a number of different kernels like R. The easiest way to set up your computer to run Jupyter for this purpose is to download and install Anaconda. However, if you need to set up Jupyter from the base Python installation, you will have to install the following modules:

  • Jupyter
  • scipy modules:
    • ipython
    • sympy
    • numpy
    • matplotlib
    • pandas

You can do so using the "pip install " command from the command prompt (Windows) or terminal (Linux) once you have Python installed.

Jupyter Notebooks (.ipynb) can be saved to a github repository and viewed online using nbviewer: https://nbviewer.jupyter.org/

Jupyter Notebooks can also be exported to formats such as HTML and posted as a webpage, however, I've yet to get .pdf exporting to work.

Data¶

Generating a mock data set. You can use this method (with high enough resolution) to generate theoretical curves. With real data, you would use Pandas or NumPy to load float arrays into the kernel.

In [23]:
xmin = 0
xmax = 10
xstep = 0.25 #Determines distance between adjacent points and hence the resolution of the graph array
dat1 = [i*xstep+xmin for i in range(int(xmax/xstep))]
dat2 = [40*sin(i) for i in dat1]
dat3 = [i**2 for i in dat1]
